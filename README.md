

# 🎯 Python Number Guessing Game

A simple beginner-friendly Python game where the player tries to guess the secret number.

## 📌 Description

This project is a basic number guessing game built with Python.  
The program asks the user to enter a number and gives hints whether the guess is too high or too low. The game continues until the correct number is guessed.

It also handles invalid inputs using exception handling.

## ✨ Features

- Guess the secret number
- Shows "too low" or "too high" hints
- Runs until the correct answer is found
- Handles non-number inputs
- Simple command-line interface

## 🛠️ Technologies Used

- Python 3

## 🧠 Concepts Practiced

- Variables
- While loops
- If / Elif / Else conditions
- User input
- Type conversion
- Try / Except error handling

## 🚀 How to Run

1. Clone this repository

```bash
git clone <your-repository-link>
```

2. Open the project folder

```bash
cd Python-Number-Guessing-Game
```

3. Run the program

```bash
python main.py
```
